Installiere Kivy auf zwei verschiedene Arten:

Mit command prompt/VSCode:
				Auf der Tastatur, drücke "win" + "R", und schreibe: "cmd", und drücke "Enter", um in Befehlsterminal 					hineinzukommen.
				Wenn Sie aus irgend einem Grund VSCode brauchen wollen, öffnen Sie VSCode, öffnen Sie einen lehren 					Codespace, und geben sie die Befehle im Terminal (Nach der Ausführung (run) vom code zu sehen) ein.
				
				Copiar + pegar diese Befehle in chronologischer Reihenfolge: 
				python -m pip install --upgrade pip setuptools virtualenv
				python -m virtualenv kvenv
				kvenv\Scripts\activate

				Öffnen Sie das Befehlsterminal mit (kvenv) und geben Sie diesen letzten Befehl im Terminal ein: 
				python -m pip install kivy[full]
